博客迁移，Jekyll到Hexo

发表于 2018-05-04 | 分类于 技术博客 |

一篇一句

对于RD而言，技术能力决定你能走多快，产品能力决定你能走多远

缘由

第一版的博客由老铁全全儿安利，用的是 Jekyll，主题是 黄玄的博客|Hux Blog。
【优点】Jekyll 文档齐全，并且是 Github Pages 推荐的静态博客系统。黄玄的主题，简洁明了。
【缺点】该主题，缺少一些功能。比如：站内搜索和文章内导航。

在博客的文章逐渐变多的时候，没有站内搜索功能，要找一篇以前写过的博客很费劲。另外，黄玄的主题需要手动在文章里面写章节号，这也是我不太喜欢的地方。

于是，在找寻了一番之后，找到了 Hexo 博客系统下的一个主题 Next。
在第三次墙裂的想迁移的时候，我终于下定决心，战胜懒癌，把博客从Jekyll 迁移到Hexo。
